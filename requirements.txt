# The coral.models.layers._get_flash_attn_func() function falls back to
# PyTorch's scaled_dot_product_attention if flash-attn is not installed,
# which on A100 dispatches to FlashAttention-2 internally via cuDNN. The
# fallback costs ~5-10% end-to-end throughput vs a standalone flash-attn install.
#
# Pin: 2.7.4.post1 is the last version ABI-compatible with torch 2.6.
# flash-attn 2.8.x uses symbols absent in torch 2.6 and crashes at import.
# Update this pin when migrating to torch 2.7+.
#
# To install (A100-only sm_80 build, ~4× faster than full build):
#     FLASH_ATTN_CUDA_ARCHS=80 pip install flash-attn==2.7.4.post1 --no-build-isolation
#
# Prebuilt wheels for torch 2.4-2.6 + CUDA 12.x are available on PyPI.
